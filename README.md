# AddressBook-NewDesign

## Overview
AddressBook-NewDesign is a console-based Address Book Application developed in C programming. The application helps users store and manage contact details such as name, phone number, email ID, and address. It supports operations like adding, viewing, searching, editing, and deleting contacts.

The project uses file handling to permanently save contact information, ensuring that data is retained even after the program is closed.

---

## Features
- Add new contacts
- View all contacts
- Search contacts
- Edit existing contacts
- Delete contacts
- Store data using file handling
- Simple menu-driven interface
